2.Описание main.py
2.1.Импортируемые библиотеки: json, requests, pprint, pandas, datetime, time, progress.Bar

2.2.class VKuser - класс для работы с фотографиями из ВКонтакте

2.2.1 Описание класса VKuser:
- url = 'https://api.vk.com/method/' - адрес для работы с методами API VK
- def __init__(self, token, version) - инициализация класса. На вход передается Токен ВК и версия API
- def get_photo_profile(self) - метод для получения информации о фотогафиях с профиля пользователя
- def get_info_photo(self, list) - метод получения конкретной информации о фотогорафиях (наименование, длина, ширина, размер, адрес). На вход получает словарь с информацией о фотографиях
- def file_name(self, list) - метод формирования уникального имени фотографии. Имя получается путем сложения строк количества лайков и даты в формате Юникод. На вход передается инфрмация о лайках и дате загрузки фотографии
- def create_file_json(self) - метод формирования и сохранения файла с информацией о фотографиях в формате json
- def sort_json(self, files_dict) - метод сортировки фотографий из списка словарей по длине и ширине. На вход получает несортированный списсок словарей.

2.3.class YD_user - класс для работы с Яндекс диском
2.3.1 Описание класса YD_user:
-  def __init__(self, token: str) - инициализация класса. На вход передается Токен Яндекс Диска
- def check_directory(self) - метод проверки наличия папки с аналогичным именем в директории Яндекс диска
- def check_photo(self, file_name) - метод проверки повторящихся фотографий. На вход получает имя фотографии
- def upload(self,n) - метод загрузки фото на Яндекс Диск, n - количество фотографий для загрузки на диск (по-умолчанию, 5 шт.)

2.4.Запрос информации у пользователя:

- Yandex_token = input('Введите токен с полигона ЯндексДиск: ') - тебуется ввести токен от Яндекс диска
